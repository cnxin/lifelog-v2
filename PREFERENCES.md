# 人员喜好与禁忌档案功能说明

LifeLog Flutter 版本支持为每个人员记录详细的喜好和禁忌档案，帮助你更好地了解身边重要的人。

## 📋 功能概述

### 喜好档案
记录人员喜欢的事物，支持自由分组和多项内容。

**使用场景**：
- 记录朋友喜欢的颜色、食物、电影类型
- 记录家人的兴趣爱好
- 准备礼物时快速查看对方喜好

### 禁忌档案
记录人员不喜欢或需要避免的事项。

**使用场景**：
- 记录过敏食物
- 记录禁忌话题
- 记录不喜欢的事物

---

## 🎯 数据结构

### PreferenceGroup（偏好分组）

```dart
class PreferenceGroup {
  final String category;      // 分类名称，如"颜色"、"食物"
  final List<String> items;   // 项目列表，如["蓝色", "黑色"]
}
```

### Person 模型字段

```dart
class Person {
  final List<PreferenceGroup> preferences;  // 喜好档案
  final List<PreferenceGroup> dislikes;     // 禁忌档案
}
```

## 📝 使用方法

### 1. 添加喜好/禁忌分类

在人员编辑页面：

1. 找到「喜好」或「禁忌」区域
2. 点击右上角的 **+** 按钮
3. 输入分类名称（如"颜色"）
4. 输入项目，用逗号分隔（如"蓝色, 黑色, 白色"）
5. 点击「添加」

### 2. 向已有分类添加项目

1. 找到对应的分类卡片
2. 点击卡片底部的 **+** 按钮
3. 输入项目名称
4. 点击「添加」

### 3. 删除项目

- 点击项目标签右侧的 **×** 按钮
- 如果分类下所有项目都被删除，分类也会自动删除

### 4. 删除整个分类

- 点击分类卡片右上角的 **×** 按钮

## 💾 数据存储

### 数据库映射

由于 SQLite 表结构限制，喜好和禁忌档案复用了现有字段：

| Person 字段 | 数据库字段 | 存储格式 |
|------------|-----------|---------|
| `preferences` | `tags` | JSON 数组 |
| `dislikes` | `photos` | JSON 数组 |

### JSON 格式

```json
[
  {
    "category": "颜色",
    "items": ["蓝色", "黑色", "白色"]
  },
  {
    "category": "食物",
    "items": ["火锅", "烧烤", "寿司"]
  }
]
```

### 兼容性

支持从 React 版本导入数据，自动解析以下格式：
- JSON 数组格式（标准）
- 字符串格式：`颜色:蓝色,黑色|食物:火锅,烧烤`

## 🔧 技术实现

### 组件结构

#### `_GroupEditor`（编辑器组件）

**位置**：`lib/pages/person_form_page.dart`

**功能**：
- 显示喜好/禁忌分类列表
- 添加新分类
- 向分类添加项目
- 删除分类或项目

**参数**：
```dart
_GroupEditor({
  required String title,           // 标题："喜好" 或 "禁忌"
  required IconData icon,          // 图标
  required List<PreferenceGroup> groups,  // 数据列表
  required AppColors colors,       // 主题颜色
  bool isDislike = false,          // 是否为禁忌（影响颜色）
  required VoidCallback onChanged, // 数据变更回调
})
```

#### `_PreferenceTags`（显示组件）

**位置**：`lib/pages/person_detail_page.dart`

**功能**：
- 只读显示喜好/禁忌档案
- 按分类分组展示
- 标签样式渲染
